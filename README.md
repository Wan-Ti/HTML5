# HTML5

## 有话说

前端的核心：URL+HTTP+HTML；MDN查询HTML相关资料；

```
HTML5技术集合 

· 新标签、新属性；
· 新的通信技术：WebSockets、WebRTC等；
· 离线存储技术：LocalStorage、断网检测；
· 多媒体技术：视频、音频；
· 图像技术：Canvas、SVG、WebGl;
· Web增强技术：HistoryAPI、全屏；
· 设备相关技术：摄像头，触摸屏；
· 新的样式技术：CSS3新的Flex、Grid布局方式
```
